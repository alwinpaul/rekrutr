# Rekrutr Dashboard
### A Candidate and Job tracking system

## Features
  - A recruiter can add new Candidate to the database
  - View list of all availabe candidate in the database
  - Filter Candidates based on selected criteria
  - A admin can add or remove new/existing recruiters
  - A admin can add or remove skills, industry verticals and other custom data which will be prefilled in frontend
  - Only an admin can remove a candidate from the database

## Technology Stack
   **Frontend**
   1. React
   2. Redux
   3. Redux toolkit
   4. Tailwind CSS
   5. Typescript

   **Backend**
1. Node
2. Express.js
3. Typescript

**Databse**
1. MongoDB
